Creation

Purpose

Given the ease of access internet provides and the enjoyable experience K-Drama delivers, I thought to provide a gateway people who are hungry for Korean historical lesson on a more enjoyable platform. Thus, this website renders a simple summary of figures who I consider as key recognizable person of Choson, and dramas viewers can watch as they binge watch through Korean Sageuk Dramas.
** IMPORTANT **
Please be aware that listed Saguek Dramas are not complete and correct recreation of history but a modified enjoyable version.

Underlying Concerns of Building This Project

Upon research, I've learned that the acceptable standard for webpage our society considers as a norm are responsibe websites that is situated to all screens. What that means is that, due to the increase in variety of monitor or visual hardware sizes, the web page has to be situated to adjust to any size of visual hardwares. Therefore, by resizing the window you, the viewer, are currently using you can witness that the elements in this webpage is correctly allocated as the size of the window changes

Technologies Used

Technologies used to build this website over the course of 40+ hours are as followed:
* Github: Used to store the code base of this webpage and to deploy this web page
* Javascript: Used to provide access for github webhosting technology to visually render this page
* HTML5: layout general information and navigatioal structure of this web page
* CSS * SASS: Used provide styling for this webpage and logical instructions for page elements to relocate as size of window changes
